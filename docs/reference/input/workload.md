# Workloads

## WorkloadSpec { #workloadspec }

Specification describing a workload trace that is replayed against the topologies.

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `pathToFile` | String | **yes** | Path to the workload trace directory (a set of Parquet files). |
| `type` | [WorkloadTypes](#workloadtypes) | **yes** | The type of workload contained in the trace. See `WorkloadTypes`. |
| `sampleFraction` | Double | no | Fraction of tasks to sample from the trace. Must be greater than 0. |
| `submissionTime` | String? | no | Optional absolute time (ISO-8601) to align the first task submission to. |
| `deferAll` | Boolean | no | Whether every task should be deferred, overriding the submission times of the trace. |
| `scalingPolicy` | [ScalingPolicyEnum](#scalingpolicyenum) | no | Policy that decides how a task's remaining work is scaled when it does not receive its full demand. See `ScalingPolicyEnum`. |
| `name` | String | no |  |
